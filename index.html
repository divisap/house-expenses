Connect Google Sheets

Follow these steps once — then every submission saves automatically to your shared sheet.

1
Go to sheets.google.com → create a new blank spreadsheet. Name it House Expenses.
2
In the spreadsheet, go to Extensions → Apps Script. Delete everything in the editor and paste the script from the SETUP_GUIDE.txt file (provided alongside this app).
3
Click Deploy → New deployment. Choose type Web app. Set Execute as = Me, Who has access = Anyone. Click Deploy and copy the URL.
4
Paste the Web App URL below and click Connect.
The URL is saved in your browser — you only need to do this once.
